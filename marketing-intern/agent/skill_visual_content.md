# Skill: Visual Content Creation

When creating event or campaign visuals, make sure the output includes:

1. Main Headline
- Short
- Attention-grabbing
- Relevant to the campaign

2. Supporting Text
- Clarifies the value of the event or promotion

3. Event Info Placement
- Date
- Time
- Venue / platform
- Registration method

4. Visual Direction
Describe:
- layout
- mood
- style
- iconography or imagery
- balance between text and graphics

5. Image Generation
If image generation is supported:
- create a clean and professional campaign visual
- make it usable as an event poster or social media graphic

If image generation is not supported:
- create a highly detailed image prompt for image generation
- specify aspect ratio and use case

## Preferred Output Types
- Poster image
- Social media square post
- Story / portrait poster
