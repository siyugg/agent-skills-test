# Required Output Files

Write the following files into `outbox/`:

1. `summary.md`
   - Overall summary of the inbox content

2. `implementation_plan.md`
   - Step-by-step implementation plan

3. `api_design.md`
   - Suggested API endpoints

4. `risks_and_questions.md`
   - Missing information, risks, and questions for the team lead

5. `junior_dev_tasks.md`
   - Clear task breakdown for a junior developer
