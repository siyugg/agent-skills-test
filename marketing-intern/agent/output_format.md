# Required Output Files

Write the following files into `outbox/`:

1. `campaign_summary.md`
- Brief summary of campaign purpose and direction

2. `taglines.md`
- 5 suggested campaign taglines

3. `social_caption.md`
- Promotional caption suitable for LinkedIn / Instagram

4. `poster_copy.md`
- Final text content to place on the event poster

5. `creative_direction.md`
- Visual direction, layout ideas, and design notes

6. `image_prompt.txt`
- Detailed prompt for generating the campaign image

7. `event_poster.png`
- Generate this if image generation capability is available

8. `social_post_square.png`
- Generate this if image generation capability is available
