# User Stories

## Patient

As a patient, I want to search for doctors by specialty so that I can find the right doctor quickly.

As a patient, I want to book an appointment so that I can reserve a consultation slot.

As a patient, I want to cancel an appointment so that I can free up the slot if I cannot attend.

## Admin

As an admin, I want to add doctors so that patients can book appointments with them.

As an admin, I want to update doctor availability so that the booking system shows correct slots.
