# Asset Notes

## Image Preferences
The image should be suitable for:
- event poster
- internal campaign use
- LinkedIn post
- social media campaign

## Preferred Formats
- Portrait poster
- Square social media graphic

## Avoid
- childish illustration style
- overly futuristic sci-fi look
- cluttered layout
- too much small text
- dark and gloomy tone

## Important
The final output should feel polished enough for actual campaign/event usage.
If image generation is available, generate the visual files directly.
If not, provide a highly usable prompt and creative layout specification.
